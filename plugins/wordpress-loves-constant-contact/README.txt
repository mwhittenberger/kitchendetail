Utilize the Constant Contact API to schedule an email campaign to let your Constant Contact subscribers know about your latest blog post. A Constant Contact account is required.

== Description ==

After writing a killer blog post, you need to let your subscribers know about it. Utilizing native WordPress accounts to collect emails and send promo blasts can be a lot of trouble. Some WordPress website owners don't feel comfortable letting visitor register for accounts on their site. Lot's of times, your promo emails get caught in spam folders.
That is why most smart email lists owners utilize third party services like Constant Contact or Mailchimp. That's where this plugin comes in.

WordPress Loves Constant Contact is currently the only WordPress plugin available that allows you to send a promo email blast to your Constant Contact subscribers from the WordPress backend. In fact, this plugin allows you to schedule the campaign from the same screen used to write the blog post.

This plugin utilizes the Constant Contact API to allow these two new best friends, WordPress and Constant Contact to communicate directly.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install the plugin in the WordPress backend or upload the unzipped plugin folder to your WordPress wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sign in or create a free account here: <a href="https://constantcontact.mashery.com/" target="_blank">https://constantcontact.mashery.com/</a>.
4. After verifying your email and signing in, register your application here: https://constantcontact.mashery.com/apps/register. Enter whatever you want for the application name and ignore all the other fields.
5. Copy your new API key and enter it into the Settings page for WordPress Loves Constant Contact (via the plugins screen or under the WordPress Settings link.
6. Hit the Save your setting button.
7. Follow the specified link in step 2 to get your Constant Contact token. You'll be asked to create or sign into your Constant Contact account.
8. After signing in, you'll be asked if you want to allow the plugin access, hit yes.
9. You'll be presented with your token, copy it and paste it into the WordPress Loves Constant Contact settings page.
10. Edit any post and you'll see the plugin section below the Editor section. Verify that the auto-filled information is correct, choose your email list, and hit the send button.
11. A new campaign for promoting your blog post will be created within Constant Contact.

== Frequently Asked Questions ==

= Do I have to have an active Constant Contact account. =

Yes, to use the plugin, you must have an active Constant Contact account.

= Is the account that I created to get my API key the same as my Constant Contact account? =

They are two different accounts.
